📦 Sistema de Controle de Estoque

Sistema de gerenciamento de estoque desenvolvido para praticar conceitos de desenvolvimento Back End utilizando FastAPI, Python e SqlAlchemy.

🚀 Funcionalidades
Cadastro de usuários
Autenticação com JWT
Login seguro com senha criptografada
Cadastro de produtos
Controle de entrada de estoque
Controle de saída de estoque
Consulta de produtos cadastrados
API REST documentada automaticamente pelo Swagger
🛠️ Tecnologias Utilizadas
Backend
Python
FastAPI
SQLAlchemy
Alembic
JWT Authentication
Passlib (Argon2)
Versionamento
Git
GitHub


⚙️ Como Executar o Projeto
Clonar o repositório
git clone [https://github.com/seu-usuario/controle-estoque.git](https://github.com/souzakeslem-maker/Estoque-API/new/main?filename=README.md)
Entrar na pasta
cd controle-estoque
Criar ambiente virtual
python -m venv venv
Ativar ambiente virtual

Windows:

venv\Scripts\activate
Instalar dependências
pip install -r requirements.txt
Executar o servidor
uvicorn main:app --reload
